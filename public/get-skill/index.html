What the Claude Code Skill Does

Fetches live docs Reads cetus.cenvero.org/docs/ at session start so it always has the latest reference.
Correct HTML structure Knows all required data attributes on root div and every clip. No more validation errors.
Full CLI reference Runs cetus context to get every flag for the exact installed version.
GSAP seek rules Enforces absolute positions, paused timelines, window.__timelines registration, and duration pinning.
Auto-updates Cetus checks the remote skill for changes every 10 minutes and updates your global install silently.
Debugging guide Black screen, frozen frames, invisible elements — the skill knows what to check and how to fix it.
