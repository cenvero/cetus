# Security Policy

## Supported Versions

Cetus supports the latest stable release published at
https://cetus.cenvero.org/manifest.json.

## Reporting Vulnerabilities

Report security issues privately to the maintainers through GitHub security
advisories for `cenvero/cetus`.

Please include:

- Affected version
- Platform
- Reproduction steps
- Impact
- Any relevant composition HTML or command line flags

Do not open public issues for vulnerabilities before disclosure is coordinated.
